# CSE_Hub 🚀

A modern, hacker-themed educational platform designed specifically for Computer Science Engineering students. CSE_Hub provides easy access to study materials, notes, and resources for various CSE subjects.

## 🌟 Features

- **Terminal-Inspired Design**: A unique, hacker-themed interface that makes learning more engaging
- **Matrix Background Animation**: Dynamic background effect inspired by "The Matrix"
- **Responsive Layout**: Works seamlessly on all devices
- **Smart Search**: Quick search functionality to find subjects and resources
- **Organized Content**: Well-structured subject materials by semester
- **Modern UI Components**: Interactive cards, hover effects, and smooth animations

## 📚 Available Subjects

### Semester 1
- Basics of Electrical and Electronics Engineering (EEUD101)
- Engineering Mathematics - 1 (MSUD101)
- Engineering Physics - 1 (PSUD101)
- English (HS101)
- Programming in C (CEUC101)

### Semester 2
- Engineering Mathematics - 2 (MSUD102)
- Engineering Physics - 2 (PSUD102)
- Object Oriented Programming (CEUC102)
- Digital Electronics (CEUS101)
- Elements of Engineering (MEUD101)
